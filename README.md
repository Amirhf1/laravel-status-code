# Laravel Status Codes

A simple Enum to return correct status codes for http responses.

## Installation

Using composer:

```bash
composer require amirhf/laravel-status-code
```

You are then free to use it as needed within your projects.

### Testing

```bash
./vendor/bin/phpunit  
```
